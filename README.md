# Adli Asistan PRO ⚖️

**Adli Asistan PRO**, yargı çalışanları (Katip, Yazı İşleri Müdürü, Hakim) için özel olarak geliştirilmiş, UYAP iş akışına tam uyumlu, modern ve "Premium" bir iş akış otomasyon sistemidir.

![Adli Asistan PRO](https://raw.githubusercontent.com/heysamy/Adli-Asistan-Pro/main/frontend/public/bg-premium.png)

## 🚀 Öne Çıkan Özellikler

- **Mila AI Akıllı Asistan:** Dosyalarınızı analiz eden ve vadesi yaklaşan işleri size fısıldayan akıllı dosya asistanı.
- **Otomatik Vade Takibi:** Senaryo şablonları sayesinde sükut sürelerini ve iş teslim tarihlerini otomatik hesaplar.
- **Premium Dashboard:** Canlı istatistikler, geciken işlerin özeti ve modern grafikler.
- **Parallax Görsel Tasarım:** Mouse hareketine duyarlı, derinlik hissi veren yüksek kaliteli arayüz.
- **Dosya ve Görev Yönetimi:** Yıl/Esas yapısına tam uyumlu dosya kaydı ve gelişmiş görev takip listeleri.
- **Güvenli Oturum:** "Beni Hatırla" destekli modern giriş ve kayıt sistemi.

## 🛠️ Teknoloji Yığını

- **Backend:** FastAPI (Python), SQLAlchemy, SQLite
- **Frontend:** React (TypeScript), Vite, Tailwind CSS, Framer Motion
- **İkonlar:** Lucide React
- **Bildirimler:** React Hot Toast

## 📦 Kurulum

### 1. Gereksinimler
- Python 3.10+
- Node.js 18+

### 2. Backend Kurulumu
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### 3. Frontend Kurulumu
```bash
cd frontend
npm install
npm run dev
```
